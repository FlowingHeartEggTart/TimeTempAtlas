# 🌡️ TimeTempAtlas: Global Temperature Anomaly Data Service

## 🌟 What is TimeTempAtlas?
TimeTempAtlas is a high-performance RESTful API service built on top of the Sanic Framework to provide rapid access to and analysis of annual historical data on global temperature anomalies. Users can query temperature data for a specific year range through a simple API interface, and support multiple data format outputs, including JSON, XML, and CSV. In addition, TimeTempAtlas provides a powerful sorting feature that allows you to sort results in ascending and descending order by temperature.

## ✨ Features
- **Flexible Querying**: Retrieve temperature anomaly data for any given year range.
- **Multiple Output Formats**: Support for JSON, XML, and CSV formats.
- **Sorting Capability**: Sort results by temperature in ascending or descending order.
- **High Performance**: Leveraging the asynchronous capabilities of the Sanic framework.

## 🛠️ Getting Started
### Prerequisites
- Python 3.x
- Sanic framework

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/FlowingHeartEggTart/TimeTempAtlas.git
   cd TimeTempAtlas
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the service:
   ```bash
   python app.py
   ```

## 📚 API Documentation
### Query Temperature Data
- **Endpoint**: `GET /query`
- **Parameters**:
  - `start`: Start year (default: 1880)
  - `end`: End year (default: 2022)
  - `sort`: Sort order (asc or desc, default: asc)
  - `format`: Output format (json, xml, or csv, default: json)

# 🌡️ TimeTempAtlas：全球气温异常数据服务

## 🌟 什么是TimeTempAtlas？
TimeTempAtlas 是一个基于 Sanic 框架构建的高性能 RESTful API 服务，旨在提供快速访问和分析全球气温年度异常历史数据。用户可以通过简单的 API 接口查询特定年份范围内的气温数据，并支持多种数据格式输出，包括 JSON、XML 和 CSV。此外，TimeTempAtlas 还提供了强大的排序功能，支持按气温升序和降序排列结果。

## ✨ 特点
- **灵活查询**：检索任何给定年份范围的气温异常数据。
- **多种输出格式**：支持 JSON、XML 和 CSV 格式。
- **排序功能**：按气温升序或降序排序结果。
- **高性能**：利用 Sanic 框架的异步能力。

## 🛠️ 开始使用
### 前提条件
- Python 3.x
- Sanic 框架

### 安装
1. 克隆仓库：
   ```bash
   git clone https://github.com/FlowingHeartEggTart/TimeTempAtlas.git
   cd TimeTempAtlas
   ```
2. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```
3. 运行服务：
   ```bash
   python app.py
   ```

## 📚 API 文档
### 查询气温数据
- **端点**：`GET /query`
- **参数**：
  - `start`：开始年份（默认：1880）
  - `end`：结束年份（默认：2022）
  - `sort`：排序顺序（升序或降序，默认：升序）

# 🌡️ TimeTempAtlas：全球的温度異常データサービス

## 🌟 什么是TimeTempAtlas？
TimeTempAtlasは、高性能なRESTful APIサービスをSanicフレームワークで構築し、全球の温度異常データへの迅速なアクセスと分析を提供します。ユーザーはシンプルなAPIインターフェースを通じて特定の年間の温度データを照会し、JSON、XML、CSVなどの複数のデータフォーマットをサポートしています。さらに、TimeTempAtlasでは温度による昇順または降順の結果を並べ替える強力なソート機能も提供しています。

## ✨ 特徴
- **柔軟な照会**：任意の年間の温度異常データを照会します。
- **複数の出力フォーマット**：JSON、XML、CSVフォーマットをサポートします。
- **ソート機能**：温度による昇順または降順での結果を並べ替えます。
- **高性能**：Sanicフレームワークの非同期能力を利用します。

## 🛠️ 使い方
### 前提条件
- Python 3.x
- Sanicフレームワーク

### インストール
1. レポジトリをクローン：
   ```bash
   git clone https://github.com/FlowingHeartEggTart/TimeTempAtlas.git
   cd TimeTempAtlas
   ```
2. 依存関係をインストール：
   ```bash
   pip install -r requirements.txt
   ```
3. サービスを実行：
   ```bash
   python app.py
   ```

## 📚 APIドキュメント
### 温度データを照会
- **エンドポイント**：`GET /query`
- **パラメーター**：
  - `start`：開始年（デフォルト：1880）
  - `end`：終了年（デフォルト：2022）
  - `sort`：ソート順序（昇順または降順、デフォルト：昇順）
  - `format`：出力フォーマット（json、xml、または csv、デフォルト：json）
